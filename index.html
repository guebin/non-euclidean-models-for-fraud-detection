About This Page

This page contains supplementary tables for the paper "Non-Euclidean Models for Fraud Detection". Methods with subscript G indicate graph-enhanced versions using GCN embeddings. Highlighted rows show graph-enhanced models for easy comparison.

A1. Undersampling Experiments

We evaluated the robustness of our proposed method across various undersampling ratios while keeping the test set unchanged.
